7월 프로젝트 기획안

프로젝트 명:빨래도우미
문제정의:
- 1인 가구/맞벌이 가구는 빨래 시점(세탁, 건조, 분리수거, 다림질)을 자주 놓쳐 생활 동선이 꼬임.
- 날씨/습도/미세먼지에 따라 세탁물 건조 효율이 달라지지만, 사용자 경험에 의존해 비효율이 발생함.
- 의류별 세탁 방법을 몰라 옷감 손상, 이염, 수축 등의 문제가 반복됨.

핵심기능:
- 세탁 일정 관리: 세탁/건조/정리 체크리스트와 알림 제공.
- 날씨 기반 세탁 추천: 당일 기상 정보(강수 확률, 습도)를 기반으로 실내/실외 건조 가이드 제공.
- 의류 케어 가이드: 옷감/세탁 라벨 기준으로 세탁 코스, 물 온도, 건조 방법 추천.
- 세제/유연제 재고 알림: 소모량 기록을 통해 구매 시점 리마인드.
- 가족/룸메이트 공유: 공동 세탁 일정과 완료 상태를 공유해 중복 작업 방지.

기대효과:
- 빨래 누락 및 지연 감소로 생활 루틴 안정화.
- 의류 손상 감소로 의류 수명 연장 및 비용 절감.
- 날씨 기반 의사결정으로 건조 시간 단축.
- 가구 구성원 간 가사 분담 명확화로 스트레스 완화.

핵심화면:

1) 홈 대시보드 (메인 화면)
   - 구성: 헤더(날씨 위젯) → 오늘의 추천 카드 → 진행 중 작업 영역 → 빠른 시작 버튼
   - 주요 요소:
     • 헤더: 좌측 앱 로고/메뉴 버튼, 중앙 화면 제목, 우측 설정/알림 벨 아이콘
     • 현재 날씨 & 습도 표시 (실시간) - 상단 헤더 아래
     • "오늘 세탁하기 좋은 날!" 추천 카드 (기상 정보 기반) - 상단
     • 진행 중인 작업 상태 (세탁 중 → 건조 중 → 정리 대기) - 중간
     • 다음 예정된 세탁 일정 미리보기 - 중하단
     • 빠른 시작 버튼 (오늘 세탁 시작) - 하단 고정
   - 사용자 액션: 추천 카드 클릭 → 세탁 일정 생성, 헤더 클릭 → 상세 기상 정보

2) 일정 캘린더 (계획 화면)
   - 구성: 월/주 전환 탭 → 캘린더 그리드 → 상세 일정 리스트
   - 주요 요소:
     • 헤더: 좌측 앱 로고/메뉴 버튼, 중앙 화면 제목, 우측 설정/알림 벨 아이콘
     • 월간/주간 뷰 전환 버튼 - 상단 좌측
     • 세탁 예정일 표시 (색상 코딩: 예정 | 진행 중 | 완료) - 중간 캘린더
     • 각 날짜별 세탁 단계 시각화 (아이콘) - 캘린더 날짜 위
     • 알림 설정 버튼 (세탁 시작 시간, 건조 시작 시간) - 상세 일정 우측
     • 반복 설정 (매주/격주/월 1회) - 상세 일정 하단
   - 사용자 액션: 날짜 클릭 → 상세 일정 추가/수정, 알림 아이콘 클릭 → 푸시 알림 설정

3) 의류 케어 가이드 (맞춤 세탁법 화면)
   - 구성: 옷감/라벨 입력 영역 → 세탁 방법 카드 → 단계별 안내
   - 주요 요소:
     • 헤더: 좌측 앱 로고/메뉴 버튼, 중앙 화면 제목, 우측 설정/알림 벨 아이콘
     • 옷감 선택 드롭다운 - 상단 좌측
     • 세탁 라벨 번호 입력 필드 또는 이미지 업로드 - 상단 중앙
     • 세탁 추천 정보 카드 (물 온도/세탁 코스/건조 방법/주의사항) - 중간
     • 단계별 아이콘 안내 (분류 → 세탁 → 헹굼 → 건조) - 중하단
     • 저장 버튼 - 하단 좌측
   - 사용자 액션: 옷감 선택 → 즉시 가이드 표시, 저장 버튼 → 자주 입는 옷 목록에 추가

4) 재고 관리 (소모품 추적 화면)
   - 구성: 세제/유연제 인벤토리 카드 → 사용량 입력 → 구매 알림 영역
   - 주요 요소:
     • 헤더: 좌측 앱 로고/메뉴 버튼, 중앙 화면 제목, 우측 설정/알림 벨 아이콘
     • 제목 & [+ 추가] 버튼 - 상단 우측
     • 현재 보유 제품 리스트 (세제, 섬유유연제, 표백제 등) - 중간 (카드 형태)
     • 각 제품별:
       - 남은 양 표시 (남음 수량 / 전체 용량) - 카드 상단
       - 진행률 바 (시각적 표시) - 카드 중상
       - 최근 사용 날짜 & 예상 소진 날짜 - 카드 중하
       - 사용 +/- 버튼 (빠른 입력) - 카드 하단
     • 구매 알림 설정 - 하단
   - 사용자 액션: 사용 버튼 클릭 → 수량 감소 기록, 구매하기 클릭 → 온라인 쇼핑 연동

5) 공유 보드 (가족/룸메이트 협업 화면)
   - 구성: 구성원 목록 → 공유 일정 → 할 일 배정 → 완료 체크
   - 주요 요소:
     • 헤더: 좌측 앱 로고/메뉴 버튼, 중앙 화면 제목, 우측 설정/알림 벨 아이콘
     • 가구 구성원 프로필 (이름, 아이콘, 상태) - 상단
     • [+ 초대] 버튼 - 상단 우측
     • 이번 주 세탁 일정 제목 - 중상단
     • 개인별 담당 작업 배정 영역:
       - 날짜 표시 (월, 7월 8일) - 카드 최상단
       - 할 일 제목 - 카드 좌측
       - 담당자 이름 - 카드 좌측
       - 예정 시간 - 카드 좌측
       - 완료 체크박스 - 카드 우측
       - 댓글/메모 - 카드 하단
     • 완료 시간 기록 - 카드 중간
     • 다음 구성원 알림 - 카드 우측
     • 수정/추가 버튼 - 하단
   - 사용자 액션: 구성원 선택 → 작업 배정, 체크박스 클릭 → 작업 완료 표시

페르소나:
- 페르소나 A(직장인, 29세, 1인 가구): 퇴근 후 빨래 타이밍을 자주 놓쳐 주말에 몰아서 처리함.
- 페르소나 B(맞벌이 부부, 34세/36세): 가사 분담은 되어 있으나 세탁 일정이 겹치거나 누락되는 문제를 겪음.
- 페르소나 C(대학생 자취생, 22세): 세탁 라벨 이해가 어려워 옷 손상 경험이 많음.

예상일정:(1달예상)
- 1주차: 문제 구체화, 유사 서비스 조사, 핵심 기능 정의, 화면 구조 설계.
- 2주차: 와이어프레임/프로토타입 제작, 사용자 시나리오 검증.
- 3주차: 핵심 기능 구현(일정/알림/가이드), 내부 테스트.
- 4주차: 피드백 반영, 완성도 개선, 최종 발표 자료 정리.
